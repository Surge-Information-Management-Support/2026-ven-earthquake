# 2026-ven-earthquake
SIMS: 2026 VEN - Earthquake

This repository is used to manage tasks related to the Red Cross/Red Crescent response to the earthquake in Venezuala.

**Links**  
  
- [IFRC GO response page](https://go.ifrc.org/emergencies/7990/details)  
- SIMS Portal page
- [Learn SIMS: standards and guidance for SIMS products](https://learn-sims.org)  
- [Task board: invite only](https://github.com/orgs/Surge-Information-Management-Support/projects/31)  
- Slack channel: invite only  
  
**Contributing**

If you are interested in contributing to tasks:

1) Set up an account on the SIMS Portal  
2) Add your Slack ID and GitHub username to your profile on the portal  
3) Join the #2026_ven_earthquake channel on the SIMS Slack, where requests will be published.
